MIPS Simulator

Simulation (with pipelining) of a MIPS R4000 CPU in Python

View on GitHub Download .zip Download .tar.gz

Welcome to the MIPS Simulator Project Site

This project was developed during my academic work at the University of Kansas School of Engineering. It is intended to demonstrate how a pipelined CPU might be simulated in code. The project was written in Python.
